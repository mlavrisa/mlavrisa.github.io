Selected Projects

Mirror Ray Tracer

A ray tracer for conic-section-derived mirrors, with 3D visualization environment written from scratch, developed in NumPy/CuPy/ModernGL.

Sleep Tracking System

A system for tracking and analyzing sleep patterns developed for Raspberry Pi and the NoIR camera module using custom signal processing techniques.

Personal Assistant App - Tidal

A personal assistant app that helps you plan out your day, stick to your habits, journal to reflect on the past, and achieve your goals.

Fourth Year Engineering Design Project

A project using plasmonic nanotiles immersed in liquid crystal between two conductive substrates to change the colour of reflected light.

Image Triangulation Interface

A user interface for image triangulation that respects edges and corners, with granular control over the result, using several image processing techniques.

4th Order Runge-Kutta Rope Simulation

A Python implementation of the 4th Order Runge-Kutta method for simulating a rope waving in the wind.
